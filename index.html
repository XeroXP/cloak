Cloak

A library to facilitate the use of Dagger in your Android app. Also sets up ButterKnife for "view injection".

For more information please see the website.

Download

Download the latest JAR or grab via Maven or Gradle:

<dependency>
    <groupId>com.f2prateek.cloak</groupId>
    <artifactId>cloak</artifactId>
    <version>(insert latest version)</version>
</dependency>

compile 'com.f2prateek.cloak:cloak:(insert latest version)'
